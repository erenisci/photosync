PhotoSync — USB-to-Cloud Photo & Video Backup
==============================================

1. Plug this USB drive into any computer.
2. Run PhotoSync (.exe on Windows, no extension on Mac/Linux).
3. On first run a setup wizard will walk you through connecting to
   your cloud (Google Drive, Dropbox, OneDrive, or S3-compatible).
4. Set a master password — this encrypts your credentials on the drive.
5. Press Start. PhotoSync scans every photo and video, skips files
   already in the cloud, and uploads the rest.

Your files, settings, and upload history all live on this drive.
No installation required.

More info: https://github.com/photosync/photosync
